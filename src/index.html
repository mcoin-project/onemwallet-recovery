Offline Usage

You can use this tool without having to be online. However balance's will not be displayed.

In your browser, select file save-as, and save this page as a file.

Double-click that file to open it in a browser on any offline computer.

Alternatively, download the file from the repository - https://github.com/mcoin-project/onemwallet-recovery

This project is 100% open-source code

Get the source code from the repository - https://github.com/mcoin-project/onemwallet-recovery. Based on code from iancoleman's bip39 repo.
